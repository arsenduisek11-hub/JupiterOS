JupiterOS v2.0 - Complete 64-bit Operating System

SUMMARY:
--------
A feature-complete, minimal 64-bit OS written in C and Assembly.
~4200 lines of code implementing a full OS with GUI, filesystems, 
multitasking, and user applications.

BUILD:
------
make              # Compile kernel
make run          # Run in QEMU
make run-kvm      # Run with KVM (faster)
make run-vesa     # Run with graphics
make iso          # Create ISO image
make clean        # Clean artifacts

COMPONENTS:
-----------
Boot       - Multiboot bootloader, long mode setup
Kernel     - 30+ modules (IDT, GDT, paging, scheduler, drivers)
Library    - Standard C functions
User Apps  - Shell, text editor, file manager

FEATURES:
---------
✓ 64-bit kernel (x86-64 architecture)
✓ Memory management with bitmap allocator
✓ Virtual memory with 4-level paging
✓ Interrupt handling (IDT, PIC)
✓ Multitasking (Round Robin scheduler, 1000 Hz)
✓ File system (FAT32 support)
✓ Graphics (VESA framebuffer, 1024x768)
✓ Input devices (PS/2 keyboard and mouse)
✓ System calls for user programs
✓ GUI with window management
✓ Device drivers (disk, keyboard, mouse, timer, RTC, serial)
✓ QEMU/KVM support (VirtIO)
